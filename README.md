# SRT マージャー（merge_srt.py）

日本語SRTファイルの字幕ブロックを整形するツール。

---

## インストール

```bash
pip install fugashi unidic-lite srt kanjize
```

---

## 基本的な使い方

```bash
python merge_srt.py input.srt [output.srt] [オプション]
```

| 引数 | 説明 |
|---|---|
| `input.srt` | 入力ファイル（必須） |
| `output.srt` | 出力ファイル（省略時は自動命名） |

出力ファイルを省略した場合、`入力ファイル名_merged.srt`（ダミーモード時は`_dummy.srt`）として保存されます。

---

## オプション

| オプション | 説明 |
|---|---|
| `--preview` | 先頭30ブロックのみ表示し、ファイルに保存しない |
| `--dummy` | ダミー変換モードで実行 |
| `--gap=秒数` | ダミー変換モードのグループ判定間隔（デフォルト: 10秒） |

---

## モード

### 通常モード（字幕マージ）

形態素解析を使って不自然な途中切れを検出し、前後のブロックをマージする。

**マージ条件（いずれかを満たす場合）:**

- **ルール1 — 単語の途中切れ:** 2ブロックを結合したときの形態素数が、前ブロック単体より少なくなる場合（単語が途中で分断されていた）
- **ルール2 — 先頭が助詞・助動詞:** 次ブロックの先頭形態素が助詞または助動詞で、かつ結合後の文字数が40字以内の場合

**マージ例:**

```
12: 原材料価格
13: が上がったり
→ マージ（「が」は助詞、合計9文字）
```

**後処理（マージ後に全ブロックへ適用）:**

| 処理 | 内容 |
|---|---|
| 句読点削除 | 「、」「。」「，」「．」を削除 |
| 漢数字変換 | 漢数字列を算用数字に変換（`kanjize` 使用、変換失敗箇所はそのまま残す） |
| スペース削除 | 半角・全角スペースを削除 |

**タイムスタンプ連結:**

マージ後、各ブロックのend timeを次のブロックのstart timeで上書きする（次ブロックとの間隔が10秒以上の場合は上書きしない）。

---

### ダミー変換モード（`--dummy`）

字幕テキストの内容に関わらず、連続するブロックをグループとしてまとめ、テキストを`※※※`に置き換える。

**グループ判定:**

- 連続するブロック間のstart/end間隔が`--gap`以内であれば同じグループ
- グループのタイムスタンプ: 先頭ブロックのstart ～ 末尾ブロックのend

---

## 使用例

```bash
# 通常モード（全処理・保存）
python merge_srt.py 字幕.srt 字幕_merged.srt

# 通常モード（プレビューのみ、保存しない）
python merge_srt.py 字幕.srt --preview

# ドラッグ&ドロップ（出力ファイル名は自動で「字幕_merged.srt」）
python merge_srt.py 字幕.srt

# ダミー変換モード（グループ間隔デフォルト10秒）
python merge_srt.py 字幕.srt --dummy

# ダミー変換モード（グループ間隔を3秒に変更）
python merge_srt.py 字幕.srt --dummy --gap=3

# ダミー変換モード（プレビュー）
python merge_srt.py 字幕.srt --dummy --gap=5 --preview
```
